# swipe-match-app

Проект веб-приложения для знакомств с расширенным алгоритмом подбора пользователей.
## Описание
Приложение позволяет пользователям просматривать анкеты других пользователей, ставить лайки/дизлайки (свайпы). 
Главная особенность проекта — наличие **Белых** и **Черных списков**, которые влияют на алгоритм выдачи анкет.

## Технологический стек
- **Backend:** Python, FastAPI
- **Frontend:** JavaScript (React/Vue/Vanilla)
- **Database:** PostgreSQL
- **Deployment:** Localhost (Docker)

## Основные функции
- Регистрация и авторизация пользователей
- Механика свайпов (Лайк / Дизлайк)
- Черный список: блокировка нежелательных пользователей по их ред флагам
- Белый список: приоритетный показ определенных пользователей по их грин флагам
- Алгоритм подбора: исключает уже просмотренных и заблокированных
- Система мэтчей
